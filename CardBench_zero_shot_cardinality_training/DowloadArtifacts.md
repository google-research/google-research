# CardBench artifacts

We have made available the CardBench datasets, their metadata, the sql workload
and the corresponding training query graphs.
Below you can find the links to all the artifacts.

## Datasets

The underlying data for the datasets is available in CSV files. We also provide
scripts for creating tables and loading the CSV files into a PostgreSQL
compatible database.
