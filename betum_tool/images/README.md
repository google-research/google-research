# Technical Briefing Images

This directory contains the images linked in `TECHNICAL_BRIEFING.md`.

*   `class_imbalance.png`: Visualization showing the class distribution and size
    discrepancy (tree dominance).
*   `tree_englobing_flowers.png`: Image demonstrating inaccurate tree labeling
    where the tree box englobes flowers.
*   `spoilt_black_spots.png`: Image showing the `spoilt` label associated with
    black spots on fruits.
*   `whole_tree_labeled_spoilt.png`: Image showing the entire tree accounted as
    `spoilt`.
